# TestProject
Инструкция для запуска и пользования.

Предварительные настройки:
В классе Properties, задать значения констант.
1. LOCALHOST = Это IP-адрес, который обозначает локальную машину (компьютер, на котором выполняется программа). 
Обычно используется для обращения к сервисам, запущенным на той же машине.
2. FTP_SERVER_IP =  Это IP-адрес FTP-сервера, к которому будет происходить подключение.
В данном случае, он по умолчанию равен LOCALHOST, то есть сервер находится на той же машине, где выполняется программа.
3. RESPONSE_WAIT_SEC = Это время в секундах, в течение которого программа будет ожидать ответа от FTP-сервера. 
Если в течение этого времени ответа не поступит, программа может считать, что произошла ошибка или тайм-аут.
4. FTP_SERVER_PORT = Это номер порта, через который программа будет подключаться к FTP-серверу. 
Стандартный порт для FTP — 21.
5. BUFFER_SIZE = Размер буфера (в байтах), который используется для передачи данных между клиентом и сервером.
Больший размер буфера может ускорить передачу данных, но также требует больше памяти. 
6. ACTIVE_MODE_CONNECTION_WAIT_SEC = Время ожидания (в секундах) установления соединения в активном режиме FTP. 
В активном режиме сервер сам инициирует соединение с клиентом, и этот параметр задает, как долго клиент будет ждать этого соединения.
7. FTP_SERVER_DEFAULT_FILE_PATH = Путь к файлу на сервере, который будет использоваться по умолчанию.

Подготовка к запуску:

Перед началом работы убедитесь, что FTP-сервер запущен и доступен по IP-адресу и порту, указанным в программе. 
По умолчанию используется 127.0.0.1 (локальная машина) и порт 21.


Запуск программы:
- Откройте консоль и запустите программу FtpStudentClient.

Ввод учетных данных и настроек:
- Введите логин и пароль для подключения к FTP-серверу.
- Программа предложит изменить IP-адрес, порт и путь к файлу на сервере.
- Если текущие значения устраивают, введите n (нет). Для изменения введите y (да) и укажите новые значения.

Основное меню:

После успешной авторизации будет выведено основное меню:
Меню:
1. Получение списка студентов по имени
2. Получение информации о студенте по id
3. Добавление студента
4. Удаление студента по id
   b. Вернуться в прошлое меню
   q. Завершение работы
   m. Выбрать активный/пассивный режим работы с FTP

Выберите одно из следующих действий, введя соответствующий номер или символ:

1. Получение списка студентов по имени:
- Введите 1 и затем укажите имя студента.
- Программа покажет всех студентов с данным именем.
2. Получение информации о студенте по ID:
- Введите 2 и затем укажите ID студента.
- Программа выведет информацию о студенте с указанным ID.
3. Добавление студента:
- Введите 3 и затем введите имя нового студента.
- Студент будет добавлен в список, и данные будут сохранены на сервере.
4. Удаление студента по ID:
- Введите 4 и затем укажите ID студента для удаления.
- Студент будет удален из списка, и изменения будут сохранены на сервере.
5. Выбор режима работы с FTP (активный/пассивный):
- Введите m, чтобы переключить режим работы.
- Текущий режим работы будет показан. Для изменения режима введите y, чтобы переключить его, или n для сохранения текущего режима.
6. Возврат в предыдущее меню:
- Введите b, чтобы вернуться в предыдущее меню и повторно выполнить авторизацию с новыми настройками.
7. Завершение работы:
- Введите q, чтобы завершить работу программы.
